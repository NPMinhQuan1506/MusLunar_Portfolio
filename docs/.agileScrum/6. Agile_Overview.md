# MusLunar Portfolio - Agile Overview

## 1. Agile Methodology
- Framework: Agile Kanban (Lite)
- Team: 2 members (1 Developer, 1 Designer)
- Process: Continuous Flow (Task-based execution within defined milestones)
- Key Phases:
    - Sprint 1 (MVP): Dec 08 - Dec 31
    - Sprint 2 (Full): Jan 01 - Mar 27
- Ceremonies:
    - Kick-off: Start of each major phase
    - Ad-hoc Sync: As needed via chat (Telegram)
    - Weekly Review: Progress check every Sunday
